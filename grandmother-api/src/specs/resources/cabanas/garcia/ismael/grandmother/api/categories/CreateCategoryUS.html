API Alta de categorías de gastos

Como administrador
quiero un API REST que de de alta categorías de gastos
para poder agrupar gastos por categorías.

Criterios de aceptación

  • El alta de una categoría de gasto debe quedar persistida en el sistema
  • Una categoría de gasto viene identificada por su nombre
Alta de una categoría de gasto
Cuando envío con el siguiente body { "name":"Agua" } (utilizando application/json) Entonces debería recibir HTTP/1.1 201 Created
